About Dataset

Context

This dataset contains 1200 images of flowers. The data collection is based on the data flicr, google images, yandex images. You can use this datastet to recognize plants from the photo.

Content

The pictures are divided into five classes: chamomile, tulip, rose, sunflower, dandelion. For each class there are about 200 photos. Photos are not high resolution, about 320x240 pixels. Photos are not reduced to a single size, they have different proportions!

Acknowledgements

The data collection is based on scraped data from flickr, google images, and yandex images.

About CNN model

Currently, model is train for 120 epoch with training dataset ,200 images for each catefory and achieve training accuracy of 100% means model can recognize training dataset with 100% accuracy. While validatation dataset,length of 100 images, gives validation accuracy of model 68%

how to improve model accuracy :

  • Here model is suffer from higher variance, because model training accuracy is 100% while validation accuracy is 68% percent. It works fine for training data but failed to newly unseen data.
  • One way to reduce variance is use large training dataset. here model trained on small size of dataset(200 for each categories).large dataset require more computation and longer times to train model.
  • Tuning hyper parameter of network can improve model performance.For example set large number of epoch, right choice of optimizer(SGD,rmsprop) and learning rate.
  • Use image augmentation. using zoom,rotation and shear functionalities on existing sample and generate more examples so model can learn these new features and gives better performance
  • change network topology. use wide network so cnn model can capture more abtract features
  • Use cross validation to avoid overfitting problem
